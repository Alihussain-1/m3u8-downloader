M3U8 Downloader

stream link
Right-click that link and pick "Download with IDM" (or a similar name in your extension's menu). IDM reads .m3u8 directly and grabs every segment itself.
"Download MP4" handles fMP4 streams (init segment + segments, saved directly as a valid .mp4) and AES-128 encrypted streams (decrypted before saving) as well as plain .ts streams (remuxed to .mp4 with ffmpeg, no re-encoding). If a stream uses a scheme this can't decrypt (e.g. SAMPLE-AES, DRM), it will say so instead of saving a broken file.
